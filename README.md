專案簡介：
模擬真實購物流程與使用者互動體驗，打造小型電商平台。

技術應用：
1. 採用 Vue 3 框架，利用 Vite 建置專案、Composition API 管理元件邏輯 以及 Pinia 處理狀態管理，建立出清晰且易於維護的前端架構。
2. 開發過程中深入理解 Vue 元件的生命週期與響應式原理，並實際操作 Vue Router 進行頁面路由配置、以 Axios 串接後端 API，實現畫面更新。
3. 第三方套件整合：
  ✓ Bootstrap 5：用於排版與樣式設計，提升頁面美觀。
  ✓ VeeValidate：實作表單驗證，確保使用者輸入資料的正確與安全。
  ✓ Axios：整合 HTTP 請求，串接後端 API 取得並渲染資料。

專案重構：
將原有 Options API 遷移至 Composition API，並導入 Pinia 處理狀態管理，提升程式碼的可讀性與可維護性。
