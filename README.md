## 專案簡介
模擬真實購物流程與使用者互動體驗，打造小型電商平台。

## 技術應用
- Vue 3 + Vite + Composition API + Pinia
- Vue Router 頁面路由配置
- Axios 串接後端 API
- Bootstrap 5 排版與樣式設計
- VeeValidate 表單驗證

## 專案重構
將原有 Options API 遷移至 Composition API，並導入 Pinia 進行狀態管理，提升程式碼可讀性與可維護性。
